Groupie Tracker - Édition Harry Potter

Présentation du Projet

Bienvenue sur le Harry Potter Groupie Tracker, une application web qui donne vie à l'univers magique d'Harry Potter à travers une interface interactive et conviviale. Ce projet a été développé dans le cadre du projet Groupie Tracker à Ynov Campus, démontrant la maîtrise du développement web en Go, de la consommation d'API REST et d'une architecture propre.

L'application permet aux utilisateurs d'explorer plus de 600 personnages, de parcourir la collection complète de livres et de films, de découvrir des centaines de sorts et de potions, et de gérer leurs personnages favoris. Construite entièrement avec la bibliothèque standard de Go et du HTML/CSS vanilla, ce projet montre à quel point les outils natifs peuvent être puissants lorsqu'ils sont correctement utilisés.

Aperçu Technique

Application web construite avec Go utilisant l'API Potter DB. Fonctionnalités : recherche de personnages, filtres, favoris, pagination, catalogues pour livres/films/sorts/potions.

Stack technique : Go (bibliothèque standard uniquement), HTML, CSS. Aucun framework, aucune dépendance externe.

Installation

git clone <repo-url>
cd API_HP/src/cmd
go run main.go

Accès : http://localhost:8080

Routes Principales

  • / - Accueil
  • /search - Recherche avec filtres
  • /character/{id} - Détails du personnage
  • /favorites - Liste des favoris
  • /books, /movies, /spells, /potions - Catalogues

API

Potter DB v1: https://api.potterdb.com/v1

Endpoints utilisés : /characters, /books, /movies, /spells, /potions

FAQ

Comment le projet a-t-il été décomposé ?

6 phases : Analyse & Design → Backend → Fonctionnalités de base → Fonctionnalités avancées → Gestion des erreurs → Design & Documentation

Répartition des tâches ?

Projet solo : 40% backend, 35% frontend, 15% débogage, 10% documentation

Gestion du temps ?

~80-100 heures sur 4 semaines. MVP d'abord, puis fonctionnalités bonus.

Stratégie de documentation ?

Documentation officielle Go, documentation Potter DB, MDN Web Docs, CSS-Tricks
